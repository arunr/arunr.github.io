Lets Have Launch

2013-08-28

I’ve put up LetsHaveLaunch - a simple, end-to-end launch page app - on GitHub.

LetsHaveLaunch uses Mongo, Exblockquotess, Angular and Node (MEAN - check out http://www.mean.io). I will go over how the app works in this blog post: it should be possible to understand the entire app in less than half an hour of reading the post & playing with the code, and will get you started with creating your own MEAN apps!

FYI, I extracted the basics of this app out of code that I’ve done for Twyst (www.twyst.in), and it took me a couple of hours to re-factor it into code suitable for a demo & remove the pieces specific to Twyst.

Do put up issues and submit pull requests to make the app better (especially on IE9!) - I’d love for this app to evolve over time to become a better launch page app stub.

Before we get started - I’ve kept the code on GitHub purposefully very short and have kept the structure very light, so it can be understood easily. I’ll discuss a better and improved structure for the app towards the end of this post.
