Venti Skinny Non-Fat Vanilla Macchiato

2013-09-06

Blame it on JAVA#History). If Gosling hadn’t consumed prodigious amounts of the brew while creating the language, we wouldn’t have had that name. Then, we wouldn’t have had the name JavaScript (which was definitely a better name than ECMAScript, which always sounded like it was inspired by a skin disease!). And so, we wouldn’t have had languages like Kava, CoffeeScript, IcedCoffee or Kaffeine, testing libraries like Mocha and ChaiJS or code editors like Espresso!

Yesterday, I dreamt of a future in which I would be using a programming language with an obscure coffee-themed name. Well, they say the best way to invent the future is to create it - and so - here’s introducing my new programming language, a dialect of JavaScript…
